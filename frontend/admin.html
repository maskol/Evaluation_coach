About Thresholds

Thresholds determine when a feature or story is considered to be "exceeding the acceptable time" in a workflow stage. Items that exceed these thresholds are flagged in bottleneck analysis and insights.

Current Behavior: The same threshold applies to all stages within each level (Feature/Story) unless you set stage-specific overrides below.
